## III.3.51 nop &ndash; no operation

 | Format | Assembly Format | Description
 | ---- | ---- | ----
 | 00 | `nop` | Do nothing.

#### Stack Transition:

&hellip;, &rarr; &hellip;,

#### Description:

The `nop` instruction does nothing. It is intended to fill in space if bytecodes are patched.

#### Exceptions:

None.

#### Correctness:

#### Verifiability:

The `nop` instruction is always verifiable.
