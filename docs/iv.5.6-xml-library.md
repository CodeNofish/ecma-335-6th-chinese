## IV.5.6 XML library

The XML Library is part of the Compact Profile. It provides a simple "pull-style" parser for XML. It is designed for resource-constrained devices, yet provides a simple user model. A conforming implementation of the CLI that includes the XML Library shall also implement the Network Library (see ยง[IV.5.4](iv.5.4-network-library.md)).

**Name used in XML:** XML

**CLI Feature Requirement:** None
